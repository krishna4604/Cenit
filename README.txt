Cenit website package
======================

Files in this package:
- index.html
- css/styles.css
- js/script.js
- images/logo.svg

Notes:
- High-resolution stock images are linked directly from Unsplash in the HTML (remote URLs). 
  If you want them bundled offline, download the images and place them into the 'images' folder,
  then update the <img> src paths in index.html accordingly.

Research / sources (public listings and RERA info used to populate company data):
- Housing listings for Cenit Ananta Residency and CIPL Dreams (various listing pages).
- RERA / company registration pages confirming address and contact.
See the chat response for direct citations.
